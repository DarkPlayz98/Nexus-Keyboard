# Nexus Keyboard

A clone-ready Android keyboard starter built for Termux development.

## Features in this starter
- Basic QWERTY input
- Symbols layout
- Delete repeat speed setting
- Theme selector: Light, Dark, AMOLED
- Persistent clipboard storage scaffold
- Settings screen

## Build in Termux
1. Clone this repo.
2. Open the project folder.
3. Build with Gradle.

Example:
```bash
./gradlew assembleDebug
```

## Enable the keyboard
After installing the APK, go to:
**Settings > System > Languages & input > On-screen keyboard**

## Notes
This is a clean starting point, not a copied Samsung or Gboard build.
